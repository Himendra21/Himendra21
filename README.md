## 🚀 Featured Project

### Focus API

A REST API built with Spring Boot that follows a clean layered architecture and implements secure authentication, validation, and database persistence.

### Features

- JWT Authentication
- Spring Security
- RESTful API Design
- Layered Architecture (Controller → Service → Repository)
- DTO Mapping
- Bean Validation
- Global Exception Handling
- Spring Data JPA
- Hibernate
- MySQL Integration
- Static Frontend Integration

### Tech Stack

`Java 17` • `Spring Boot` • `Spring Security` • `JWT` • `Spring Data JPA` • `Hibernate` • `MySQL` • `Maven`

🔗 **Repository:**  
https://github.com/Himendra21/focus-api
